# Changelog

All notable changes to the XPayr Node.js SDK are documented here.

## 0.1.0 - 2026-07-17

- Initial Merchant API client for payments, merchant data, networks, and webhooks.
- Constant-time HMAC-SHA256 webhook verification.
- TypeScript declarations and Node.js 18+ support.
- Automated tests, package validation, and provenance-ready release workflow.
